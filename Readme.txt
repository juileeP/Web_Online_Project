
WEB DEVELOPMENT PROJECT

==================

Author: Juilee Patankar


1. Developed a website for ordering food online through restaurants. 
2. Implemented a user login system, to view and place previous orders, store user location information, etc.
3. Implemented the shopping cart system using Java Servlets and JSP.
4. The payment system implemented using integration with Stripe API. 
5. Admin system in place for changing the restaurant info.
6. Front-end web design implemented using HTML, CSS and Javascript.
7. Contains a feedback form to give user suggestions and comments.
8. Automatic emails will be sent to restaurants, user and the owner. 
9. Database implemented using MySQL.
10. Hosted on Amazon Web Services (AWS) and Apache Tomcat.
